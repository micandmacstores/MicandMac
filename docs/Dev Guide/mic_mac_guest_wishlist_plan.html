Guest user - how it works
Guest checkout + data strategy
TypeScriptlocalStorage
Guest lands on site → Cart + Wishlist saved to localStorage
Guest clicks checkout → Shopify native guest checkout (email only, no account needed)
Guest creates account later → Merge triggered: localStorage cart + wishlist copied to Shopify customer account
Guest logs into existing account → Same merge - localStorage data syncs into account then cleared
Result: Customer sees their full history - orders + wishlist - in one profile
