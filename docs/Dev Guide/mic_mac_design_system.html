Usage rules - do and don't
Do
Use Cormorant Garamond for all headings - it carries the luxury feel from packaging
Use gold (#B8924A) for eyebrows, icons, star ratings, badges on dark backgrounds
Use burgundy (#4A1040) for primary CTAs, nav text, section accents
Use cream (#F5ECD7) and blush (#E8C8C0) for alternating section backgrounds
Keep body text #5C3D2E on pearl white - warm brown, never cold black
Always use pill-shaped buttons (border-radius: 999px)
Use generous whitespace - luxury brands breathe
Don't
Never use gold (#B8924A) as body text on light backgrounds - fails contrast at 3.1:1
Never use Inter, Roboto or Arial - use DM Sans for all UI text
Never use pure black (#000000) or pure white (#FFFFFF) - use plum and pearl
Never mix burgundy and blush heavily - keep one dominant per section
Never use forest green (#2D4A3E) from old system - the brand is burgundy/gold
Never use sharp corners on buttons - always pill or rounded (8px min)
Never use gold for error / success states - those have dedicated semantic colours
